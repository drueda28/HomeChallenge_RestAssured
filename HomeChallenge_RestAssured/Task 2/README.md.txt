
Automation: Swagger Petstore Sample

An Api tests automation have been created for 'Swagger Petstore' project

- Prerequisites
To run the test automation for Swagger Petstore project just install
Java:
Jdk: https://www.oracle.com/java/technologies/downloads
Maven: https://maven.apache.org/install.html
Install any compatible IDE with Java (NetBeans, Eclipse, IntelliJ, among others)


- Installation
To install and run the test automation for 'Swagger Petstore' project just follow below steps:

Open Cmd and run the next maven command (Make sure you execute the command in the same directory where pom.xml is located): 
	mvn package jetty:run

Note: If you want to see the Swagger page, you can type localhost:8080 on the browser

Run the automation tests
